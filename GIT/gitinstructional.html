GitHub Setup

GitHub Installation

sudo apt-get instal git

Workflow

  1. Make a repository on github.com
  2. Clone that repository to my local machine
    git clone https://github.com/b10dev/somerepo.git
  3. Make a gitignore file
    touch .gitignore
  4. Set globals
    git config --global user.name "b10"
    git config --global user.email "b10incdevelopers@gmail.com"
  5. Start tracking the files
    git status
    git add -A

Daily Order of Operations

  1. git status
  2. git commit -m "commit notes"
  3. git status
  4. git push origin master
